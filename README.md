# Landscape

Landscape allows you to create a map of your applications.

```
                 +----------+    +----------+                      
                 | Person   |    | Person   |                      
                 +---------++    +--+-------+                      
                           |        |                              
                           |        |                              
                           |        |                              
                       +---v--------v-+                            
                       |              |                            
                  +---->  Application <--------------------+       
                  |    |              |                    |       
                  |    +-----^--------+                    |       
                  |          |                             |       
                  |          |                             |       
                  |          |                             |       
                  |          |                             |       
         +--------+-----+    |  +------------+      +------+------+
         |  Component   |    |  | Component  |      |  Component  |
         |              |    +--+            |      |             |
         +-------^------+       +------------+      +-------------+
                 |                                                 
                 |                                                 
                 |                                                 
                 |                                                 
+------------+   |   +-----------+                                 
| Component  +---+---+ Component |                                 
|            |       |           |                                 
+------------+       +-----------+    
```
